# Project name : Blood Donation Application - A11

## Description : 
This repository contains the code of Blood Donation Application - A11 - Client side. 
The Blood Donation app that user from all over the country can use to connect with blood donor in any emergencies. Or a user can also be a donor to contribute in this great social work. Our purpose is to help the socity and make it beautiful.
